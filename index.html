GeoPampa

Sistema Web Fullstack feito com o objetivo de auxiliar os alunos ingressantes da instituição de ensino onde trabalho a localizar prédios, instalações e paradas de ônibus com o auxilio de um mapa interativo. Também é possível traçar rotas, calcular a distância entre dois locais além de outras funcionalidades. O sistema foi hospedado utilizando servidores EC2 da AWS e disponibilizado para a comunidade com domínio próprio, ip fixo e certificado SSL/TLS.

GeoNature

Sistema web Fullstack feito para divulgar informações sobre animais silvestres encontrados e catalogados na instituição de ensino onde trabalho. Os dados foram armazenados em uma ApiRest e utilizados na plataforma web para gerar as funcionalidades como os cards interativos contendo as informações de interesse aos usuários promovendo a união entre a tecnologia e meio ambiente apresentando uma solução robusta e escalável. Este sistema foi hospedado na AWS utilizando um servidor EC2 e distribuido para a comunidade com ip fixo, domínio adquirido na plataforma ecom certificado de segurança SSL/TLS.
